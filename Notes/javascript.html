JavaScript Notes

1. What is JavaScript?

  • JavaScript is a high-level interpreted programming language.
  • Used to create static and dynamic web applications.
  • Works with HTML and CSS to build interactive websites.
  • Can run outside the browser using Node.js.
  • Can be combined with frameworks like React and libraries.

2. Variables

  • A variable is a container that stores a value.
  • Declaration: let bookName;
  • Initialization: bookName = "Harry Potter";
  • Both: let bookName = "Harry Potter";

3. Rules to Name Variables

  • Should be unique.
  • Can use letters, numbers, _ and $.
  • Should not start with numbers.
  • Case sensitive.
  • Keywords not allowed.

4. Data Types

Primitive (Immutable):

  • String
  • Number
  • Boolean
  • BigInt
  • null
  • undefined
  • NaN
  • Symbol

Non-Primitive (Mutable):

  • Array
  • Object
  • Function

5. var, let, const

var

  • Function scoped
  • Can reassign
  • Can redeclare
  • Hoisted and initialized as undefined

let

  • Block scoped
  • Can reassign
  • Cannot redeclare
  • Hoisted but not initialized (TDZ)

const

  • Block scoped
  • Cannot reassign
  • Cannot redeclare
  • Objects/arrays values can be modified

6. Truthy and Falsy Values

Truthy:

  • All numbers except 0
  • All non-empty strings
  • Arrays
  • Objects
  • Functions
  • true

Falsy:

  • false
  • 0
  • "" (empty string)
  • null
  • undefined
  • NaN
  • 0n
  • -0

7. Type Coercion

Implicit: Automatically done by JavaScript.

"5" + 2 → "52"
"5" - 2 → 3
10 == "10" → true
10 === "10" → false

Explicit: Manually done by programmer.

Number("100") → 100
String(50) → "50"
Boolean(1) → true

8. Temporal Dead Zone (TDZ)

  • Only for let and const.
  • Time between declaration and initialization.
  • Accessing variable before initialization gives ReferenceError.

9. Scope

Global Scope: Declared outside functions and blocks.

Function Scope: Declared inside a function.

Block Scope: Declared inside {} (if, for, while).

10. Hoisting

  • JavaScript moves declarations to the top of their scope.
  • var is hoisted and initialized as undefined.
  • let and const are hoisted but not initialized (TDZ).
  • Function declarations are completely hoisted.

11. Expressions vs. Statements

Expressions

An expression is something that produces a value or returns something. If JS can calculate it and return a result, it is an expression.

  • 5 + 3 // 8
  • "Hello" + "25" // "Hello 25"
  • x * 2 // produces a value
  • Assignment Expression: a = 5 (This works inside console.log because it returns the value assigned).

Statements

A statement performs an action. It does something, but it may not return a value. Statements cannot be used where a value is required (e.g., inside console.log()).

let x = 10; // Statement: Declares a variable.
if (x > 5) { ... } // Statement: Controls flow.

  • Conditional Statements (Decision Making)

    • if, if...else, if...else if...else
    • switch
    • Ternary Operator (not a control statement)

  • Looping Statements

    Used when we need to repeat a task. Common loops include:

    • for loop: for(let i=0; i < 5; i++) { ... }
    • while, do...while
    • for...of, for...in

  • Jump Statements

    • break: Stops the loop entirely.
    • continue: Skips the current iteration and moves to the next.
    • return: Exits from a function and optionally returns a value.

Control Statements

Control statements change the normal "top-to-bottom" flow of execution. They decide which code runs, how many times, and when to stop.

12. Operators

Operators are special symbols used to perform operations on values and variables (Arithmetic, Assignment, Comparison, and Logical).

Arithmetic Operators: Used for arithmetic operations

Operator Meaning
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus (Remainder)
** Exponent
++ Increment
-- Decrement

Assignment Operators: Used to compare values

Operator Meaning
= a = 5
+= a += 5
-= a -= 5
*= a *= 5
/= a /= 5

Comparison Operators: Used to compare values

Operator Meaning
== Equal to
=== Strict Equal (checks value + type)
!= Not Equal
!== Strict Not Equal
> / < Greater than / Less than
>= / <= Greater or equal / Less or equal

Logical Operator: Used for condition

Operator Meaning
&& AND
|| OR
! NOT

Ternary Operator (Short if-else):

condition ? trueValue : falseValue

Type Operator:

typeof (e.g.,typeof 10 returns "number")
